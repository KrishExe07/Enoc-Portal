TROUBLESHOOTING:
---------------

Error: "Cannot connect to authentication server"
  → Run restart-backend.bat or start-portal.bat

Error: "Invalid email domain"
  → Use @charusat.edu.in for students
  → Use @charusat.ac.in for faculty/admin

Error: Backend won't start
  → Check if port 5000 is already in use
  → Run verify-config.bat to diagnose
